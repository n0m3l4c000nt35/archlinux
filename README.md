## Instalación

Configurar la distribución del teclado
```bash
loadkeys la-latin1
```
![loadkeys](https://github.com/user-attachments/assets/68dc09ec-f2e9-476e-bd0e-ea680a462c6d)

Para utilizar una de las fuentes más grandes
```bash
setfont ter-132b
```
